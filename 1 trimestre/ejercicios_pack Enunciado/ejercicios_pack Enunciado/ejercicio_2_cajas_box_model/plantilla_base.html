Ejercicio 2: Entendiendo el Modelo de Caja

En este ejercicio vamos a explorar cómo funciona el modelo de caja de CSS con diferentes configuraciones de box-sizing.

Caja 1: Content-box (modo por defecto)

Esta es la Caja 1

Configuración: width: 300px, padding: 20px, border: 5px solid, box-sizing: content-box

Con content-box, el width se aplica solo al contenido. El padding y el border se SUMAN al ancho total.

¿Cuál es el ancho total real de esta caja? Calcula: 300px (contenido) + 20px (padding-left) + 20px (padding-right) + 5px (border-left) + 5px (border-right) = 350px

Caja 2: Border-box (modo recomendado)

Esta es la Caja 2

Configuración: width: 300px, padding: 20px, border: 5px solid, box-sizing: border-box

Con border-box, el width incluye contenido, padding y border. El ancho total es exactamente el especificado: 300px.

¿Cuál es el ancho del contenido real? Calcula: 300px (total) - 20px (padding-left) - 20px (padding-right) - 5px (border-left) - 5px (border-right) = 250px

Caja 3: Centrada horizontalmente con margin: auto

Esta es la Caja 3

Configuración: width: 400px, padding: 30px, border: 3px solid, box-sizing: border-box, margin: 0 auto

Esta caja está centrada horizontalmente gracias a margin: 0 auto;

El navegador calcula automáticamente los márgenes izquierdo y derecho para centrar el elemento.

Nota: Para que margin: auto funcione, el elemento debe tener un width definido y debe ser de tipo display: block;

Instrucciones de Verificación

  1. Aplica los estilos CSS especificados en las instrucciones
  2. Calcula las dimensiones reales de cada caja
  3. Abre DevTools (F12) y verifica tus cálculos en la pestaña "Computed" o "Diseño"
  4. Observa cómo el navegador muestra visualmente el modelo de caja
